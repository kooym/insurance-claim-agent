## 1. 룰 체계 구조

```
청구 접수
    │
    ├─ [COM] 공통 선행 룰 ──────────────── 실패 시 즉시 부지급/보류
    │    ├─ COM-001: 계약 유효성
    │    ├─ COM-002: 면책기간
    │    ├─ COM-003: 면책사유 (KCD 기반)
    │    └─ COM-004: 중복/사기 징후
    │
    ├─ [담보별 룰] 병렬 처리
    │    ├─ SIL: 실손의료비
    │    ├─ IND: 입원일당
    │    ├─ SUR: 수술비
    │    ├─ AUTO-DIN: 자동차 대인배상
    │    └─ LAW: 변호사선임비용
    │
    ├─ [FRD] 보험사기 조사 선정 룰 (병행)
    │
    └─ [FIN] 최종 판정 및 결과 생성
```

## 2. 공통 선행 룰

### RULE-COM-001: 계약 유효성 확인
```
출처: 보험업법 제4편 일반사항

입력:
  - 사고일 (YYYY-MM-DD)
  - 계약 상태 (유효/실효/해지/무효)
  - 보험료 납입 상태 (정상/연체/납입유예/실효)

판단 로직:
  IF 계약 상태 == "유효" AND 납입 상태 IN ["정상", "납입유예기간이내"]
    → PASS
  ELSE IF 납입 상태 == "연체" AND 납입최고기간 이내
    → PASS (단, 보험금 지급 후 연체보험료 공제)
  ELSE
    → FAIL → 부지급 (사유: 계약 실효/해지)

출력:
  result: PASS | FAIL
  reason: "계약 상태 정상" | "계약 실효 - 납입유예기간 초과"
```

### RULE-COM-002: 면책기간 확인
```
출처: 현대해상 약관 제3조 (면책기간)

입력:
  - 계약일 (YYYY-MM-DD)
  - 사고일 (YYYY-MM-DD)
  - 사고 유형 ("질병" | "재해" | "암" | "자동차사고")

판단 로직:
  IF 사고 유형 == "재해" OR 사고 유형 == "자동차사고"
    → 면책기간 없음 → PASS

  ELIF 사고 유형 IN ["질병", "암"]
    면책기간 = 90일
    경과일수 = (사고일 - 계약일).days
    IF 경과일수 >= 면책기간
      → PASS
    ELSE
      → FAIL → 부지급 (사유: 면책기간 이내)
      부가안내: "재해성 사고의 경우 면책기간 없이 보장 가능, 사고 경위 재확인 필요"

출력:
  result: PASS | FAIL
  waiting_period_days: 90 | 0
  elapsed_days: 숫자
  reason: 텍스트
```

### RULE-COM-003: 면책사유 확인 (KCD 코드 기반)
```
출처: 현대해상 약관 제2조 (보험금을 지급하지 않는 사유)

입력:
  - KCD 코드 (예: K70.3)
  - 사고 경위 (자해여부, 음주여부, 고의여부)

절대적 면책 KCD 코드 목록:
  알코올 관련:
    F10.x  - 알코올 사용장애 전반
    K70.0  - 알코올성 지방간
    K70.1  - 알코올성 간염
    K70.2  - 알코올성 섬유증 및 경화증
    K70.3  - 알코올성 간경변증  ← 시나리오 3 해당
    K70.4  - 알코올성 간부전
    K70.9  - 상세불명의 알코올성 간질환

  마약/약물 관련:
    F11.x  - 아편계 사용장애
    F12.x  - 칸나비노이드 사용장애
    F13.x  - 진정수면제 사용장애
    F14.x  - 코카인 사용장애
    F15.x  - 기타 흥분제 사용장애

  자해/자살 관련:
    X60-X84 - 고의적 자해 (자살 및 자살시도)

  기타 절대적 면책:
    - 전쟁, 외란, 혁명, 내란 (진단서 상 외상 코드 + 사고 경위 확인)

판단 로직:
  IF KCD코드의 앞 3자리가 절대적 면책 코드 목록에 포함
    → FAIL → 즉시 부지급
    적용_약관조항: "제2조 제1항 제N호"

  ELIF 사고_경위에서 고의/자해/음주 확인
    → FAIL → 부지급 또는 담당자 검토 플래그

  ELSE
    → PASS

출력:
  result: PASS | FAIL
  matched_exclusion_code: null | "K70.3"
  policy_clause: null | "제2조 제1항 제3호"
  reason: 텍스트
```

### RULE-COM-004: 이상 징후 및 중복 청구 확인
```
출처: 금감원 보험사기예방 모범규준 (행2018-41002), 현대해상 공시 기준 (2024.05)

입력:
  - 청구 이력 (동일 청구건 존재 여부)
  - 가입 이력 (가입 후 단기 청구 여부)
  - 청구 패턴 (반복 청구 여부)

판단 로직:
  # 중복 청구 체크
  IF 동일_청구건_존재 == True
    → 보류 → 담당자 확인 (중복 청구 의심)

  # 단기 가입 후 청구
  IF (청구일 - 가입일).days <= 30 AND 청구_유형 IN ["수술비", "입원급여금"]
    → 담당자 검토 플래그 ON

  # 반복 청구
  IF 최근_1년_동일_유사_사고_청구_횟수 >= 3
    → 담당자 검토 플래그 ON

  ELSE
    → PASS (단, 아래 FRD 룰에서 추가 확인)
```

## 3. 실손의료비 룰

> **세대별 자기부담금 적용 기준** (⚠️ 계약 체결 시점에 따라 세대 구분 필수)

### RULE-SIL-000: 세대 판별
```
계약 체결일 기준:
  ~2009.09.30       → 1세대 (급여+비급여 80%, 자기부담금 없음)
  2009.10~2012.03   → 1세대 후기 (입원 90%, 통원 80%)
  2012.04~2021.06   → 3세대 (급여 80%+비급여 80%)
  2021.07~현재      → 4세대 (급여 80% + 비급여 특약 70%)

⚠️ 목업에서는 4세대 기준 적용 (계약일 기준으로 세대 자동 분류 필요)
```

### RULE-SIL-001: 4세대 실손의료비 지급 산정
```
출처: 실손의료보험 4세대 표준약관 (금융위원회 고시)

[입원 급여]
  지급액 = 급여_본인부담금 × (1 - 자기부담률)
  자기부담률 = 20%
  → 지급액 = 급여_본인부담금 × 80%

[입원 비급여] (비급여 특약 가입 시)
  지급액 = 비급여 × (1 - 자기부담률)
  자기부담률 = 30%
  → 지급액 = 비급여 × 70%

[통원 급여]
  지급액 = MAX(급여_본인부담금 - 공제금액, 0)
  공제금액 = MAX(급여_본인부담금 × 20%, MIN_공제)
  MIN_공제: 의원 1만원 / 상급·종합병원 2만원

[통원 비급여] (비급여 특약 가입 시)
  지급액 = MAX(비급여 - 공제금액, 0)
  공제금액 = MAX(비급여 × 30%, 3만원)

※ 4세대 연간 자기부담금 상한: 200만원 (급여+비급여 합산)
  → 상한 초과분은 100% 지급

[3세대 기준 (구계약)]
  지급액 = (급여_본인부담금 + 비급여) × 80%
  자기부담금 = MAX((급여+비급여) × 20%, 1만원) ← PoC 목업에 사용 중인 기준
```

### RULE-SIL-002: 연간 한도 적용
```
입원 연간 한도: 5,000만원 (보험연도 기준)
통원 1회 한도: 30만원 (급여+비급여 합산 지급액 기준)
통원 연간 횟수 한도: 180회

판단 로직:
  잔여한도 = 연간한도 - 금년_누적지급액
  이번_지급액 = MIN(산정_지급액, 잔여한도)
  IF 잔여한도 <= 0
    → 한도 소진 → 부지급
```

### RULE-SIL-003: 비급여 특별 처리
```
출처: 현대해상 약관 비급여 특약 조항

도수치료 (비급여):
  → 도수치료 특약 가입 여부 확인
  → 가입 시: 연간 50회 한도 내에서 70% 지급
  → 미가입 시: 기본 실손에서 70% 지급 (4세대)
  조사 선정 기준: 합리적 수준 초과 반복 청구 → FRD 플래그

비급여 주사 (영양제, 비타민 등):
  → 의학적 필요성 판단 필요
  → FRD-005 조사 기준 적용 (비급여 약제 허가범위 확인)

MRI/MRA (비급여):
  → 비급여 MRI 특약 가입 여부 확인
  → 처방전 기재 여부 확인

비급여 비중 경고 (담당자 플래그):
  비급여율 = 비급여 / (급여_본인부담 + 비급여)
  IF 비급여율 >= 60%
    → 담당자 검토 플래그 ON
    reason: "비급여 비중 {비급여율}% 초과 — 의학적 필요성 검토 요"
```

## 4. 입원일당 룰

### RULE-IND-001: 입원 인정 기준
```
출처: 현대해상 약관 제4조 (입원의 정의)

인정 조건:
  1. 의사 지시에 의한 입원
  2. 6시간 이상 의료기관 체류 (당일 퇴원 포함)
  3. 진료기록부 상 입원 기록 존재
  4. 자택 등에서 치료 불가한 사유

인정 일수 계산:
  입원 일수 = (퇴원일 - 입원일 + 1)
  → 입원 당일 1일, 퇴원 당일 1일 각각 산정
  예: 11/10 입원 ~ 11/14 퇴원 → 5일

특수 케이스:
  정신과 입원 (F 코드): 정신질환 특약 가입 여부 확인
  한방 입원: 한방 특약 가입 여부 확인
  요양병원 입원: 의학적 입원 필요성 조사 대상 (FRD-013)
```

### RULE-IND-002: 입원일당 지급액 계산
```
지급액 = 1일_지급금액 × 인정_입원일수

1일 지급금액: 계약 시 설정 금액 (예: 30,000원/일)
  ※ 병원 종류별 차등 적용 가능 (간호간병통합서비스 특약 가입 시):
     - 일반병원: 18만원/일
     - 종합병원: 19만원/일
     - 상급종합병원: 20만원/일

한도:
  1회 입원: 최대 180일
  연간 누적: 최대 365일

연간 한도 체크:
  잔여일수 = 365 - 해당_보험연도_누적입원일수
  인정_입원일수 = MIN(계산된_입원일수, 잔여일수, 180)
```

### RULE-IND-003: 특수 입원 처리
```
정신과 입원 (KCD F코드):
  → 보험증권 특약 목록에 "정신질환입원특약" 확인
  → 없으면 미지급

한방 입원:
  → 보험증권 특약 목록에 "한방입원특약" 확인
  → 없으면 미지급

요양병원 입원:
  → 기본 입원일당 적용 (단, 요양병원 특약 별도 가입 여부 확인)
  → FRD 룰 RULE-FRD-013 자동 적용 (의학적 필요성 검토)
```

## 5. 수술비 룰

### RULE-SUR-001: 수술 해당 여부 및 종류 판별
```
출처: 현대해상 약관 별표2 (수술분류표)

입력: 수술명, 수술 코드 (KCD 수술 코드 또는 약관 자체 코드)

수술 종류별 정액 지급표:
  1종: 100,000원   (단순 절개, 봉합, 발치 등)
  2종: 300,000원   (담낭 복강경, 충수 단순 등)
  3종: 500,000원   (충수 복강경, 유방 종양, 갑상선 등)
  4종: 1,000,000원 (대장 절제, 유방 전절제, 자궁 전절제 등)
  5종: 2,000,000원 (뇌 수술, 심장 수술, 폐 절제 등)

주요 수술 매핑 테이블:
  수술코드 4701 → 충수절제술 (복강경)  → 3종
  수술코드 4702 → 충수절제술 (개복)    → 3종
  수술코드 8510 → 유방 종양 절제술     → 3종
  수술코드 8520 → 유방 전절제술        → 4종
  수술코드 2100 → 갑상선 전절제술      → 3종 (C73) / 4종 (악성)
  수술코드 5100 → 담낭 절제술 (복강경) → 2종
  수술코드 9100 → 백내장 수술          → 1종
  수술코드 9200 → 치핵 수술            → 1종
  수술코드 7000 → 자궁 근종 절제술     → 3종
  수술코드 0101 → 두개골 절개 뇌수술   → 5종
  수술코드 3600 → 관상동맥 우회술      → 5종
  수술코드 4300 → 대장 절제술          → 4종

코드 미매핑 시:
  → LLM에게 수술명 기반 분류 요청 + 담당자 검토 플래그 ON
```

### RULE-SUR-002: 수술비 중복 지급 방지
```

```

### RULE-SUR-003: 비수술적 시술 제외
```

```

## 7. 변호사선임비용 룰

### RULE-LAW-001: 지급 사유 (6가지)
```
지급 트리거 (1사고당 한 가지라도 해당 시):

[형사처벌 경로]
1. 피보험자가 구속영장에 의해 구속된 경우
2. 검찰에 의해 공소제기(기소)된 경우 (약식기소 제외)
3. 약식기소 후 법원에 의해 공판절차로 전환된 경우
4. 검사 또는 피보험자가 약식명령에 불복하여 정식재판 청구한 경우

[경찰조사 단계 — 특정사고 한정]

※ "중대법규위반 교통사고" 정의:
   교통사고처리특례법 제3조 제2항 단서 해당 사고
   (신호위반, 중앙선침범, 과속, 음주운전, 무면허, 보행자 보호의무 위반 등)
```

### RULE-LAW-002: 보상 한도 계산
```

    → 가입금액 5,000만원 초과: 1,000만원 한도

[최신 개정 기준 (2025년 10월~)]
  심급별 차등 지급:
    1심: 2,000만원
    2심: 2,000만원
    3심: 1,000만원
    총 합산 5,000만원 한도
```


```
필수 서류:
  1. 청구서 (회사 소정 양식)

  지급사유 1호 (구속): 구속영장 사본 필요
  지급사유 2호 (기소): 공소장 필요

```

## 8. 보험사기 조사 선정 룰

> 출처: 현대해상 공시 기준 (2024.05.19), 금감원 보험사기예방 모범규준

### RULE-FRD 조사 선정 기준 (13개 항목)

```
FRD-001: 단기 집중 가입
  IF (청구일 - 최근_담보가입일).days <= 90 AND 청구유형 IN ["수술", "입원"]
  → 플래그 ON ("치료 목적 합리적 의심")

FRD-002: 불법 의료기관 청구
  체크: 청구 의료기관명이 당사 블랙리스트 또는 금감원 적발 기관 목록에 포함
  → 플래그 ON + 담당자 즉시 통보

FRD-003: 반복 청구
  IF 최근_1년_내_유사청구_건수 >= 3
  → 플래그 ON ("합리적 수준 초과 반복 청구")

FRD-004: 과다 비급여
  IF 비급여 / 공시된_비급여_평균 >= 2.0
  → 플래그 ON ("비급여 과다 청구 의심")

FRD-005: 비급여 약제 허가 외 사용
  해당 항목: 영양제, 비타민제, 미용시술 등
  체크: 약사법 허가사항 대비 과도한 처방 여부
  → 플래그 ON 가능

FRD-006: 수회 선결제
  IF 세부내역서상 복수회 선결제 확인
  → 플래그 ON

FRD-007: 증상 개선 평가 없는 반복 치료
  해당: 도수치료, 물리치료 등
  체크: 진료기록에 증상 개선 평가 기재 여부
  → 기재 없으면 플래그 ON

FRD-008: 당일 내원 수술
  체크: 진단서 발급일 == 수술일 (과거 치료이력 없음)
  → 플래그 ON (백내장, 갑상선, 관절경 등)

FRD-009: 낮병동 허위 입원
  체크: 입원확인서상 6시간 입원 요건 만족 여부
       (입원 시각 ~ 퇴원 시각이 6시간 미만이면서 입원 처리된 경우)
  → 플래그 ON

FRD-010: 허위 입원 / 과다 입원
  체크: 진단명 대비 입원 일수 통계적 이상값 (평균 + 2표준편차 초과)
  → 플래그 ON

FRD-011: 학회 기준 이탈 치료
  체크: 해당 KCD코드의 표준 치료 프로토콜 대비 치료 내용 이탈
  → LLM 판단 + 플래그 ON

FRD-012: 서류 불일치
  체크: 진단서 KCD코드 ↔ 세부내역서 치료 내용 불일치
       진단서 의사명 ↔ 수술확인서 의사명 불일치
  → 플래그 ON + 담당자 즉시 통보

FRD-013: 요양병원 입원 의학적 필요성
  체크: 요양병원 입원 청구 시 진단명 + 입원 사유의 의학적 타당성
  → 자동 조사 대상 지정
```

## 9. 최종 판정 룰

### RULE-FIN-001: 판정 분류 기준
```
판정: "지급"
  조건: 모든 COM 룰 통과 + 담보 룰 산정 완료 + FRD 플래그 없음
  자동처리 가능 여부: 산정액 <= 자동처리_임계값 AND FRD 플래그 없음

판정: "지급(담당자검토)"
  조건: COM 룰 통과 + 산정 완료 + FRD 플래그 있음 OR 산정액 > 임계값
  처리: 담당자가 검토 후 최종 승인

판정: "부지급"
  조건: COM 룰 중 하나라도 FAIL (면책기간, 면책사유, 계약실효)
       OR LAW 룰 면책사유 해당
  자동처리: FRD 플래그 없는 경우 자동 부지급 + 안내문 생성 가능

판정: "보류"
  조건: 필수 서류 미비 OR FRD-002 (블랙리스트 의료기관) OR 서류 불일치 (FRD-012)
  처리: 추가 서류 요청 안내문 자동 생성

판정: "일부지급"
  조건: 연간 한도 초과 OR 중복보험 비례 보상
```

### RULE-FIN-002: 자동처리 임계값 (Human-in-the-loop 기준)
