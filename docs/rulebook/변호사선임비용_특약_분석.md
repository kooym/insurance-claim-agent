## 1. 약관 원문 핵심 조항 정리

### 제1조 (보험금의 지급사유)

**① 지급 트리거 — 1사고마다 아래 중 하나 해당 시 변호사선임비용(실제 부담액) 지급**

| 번호 | 사유 | 분류 |
|------|------|------|
| 1호 | 구속영장에 의해 구속 | 형사절차 |
| 2호 | 검찰 공소제기(기소, 약식기소 제외) | 형사절차 |
| 3호 | 약식기소 후 법원이 공판절차 전환 | 형사절차 |
| 4호 | 검사/피보험자가 약식명령 불복 → 정식재판 | 형사절차 |
| 5호 | 사망·중대법규위반 → 경찰조사 후 **불송치** | 경찰단계 |
| 6호 | 사망·중대법규위반 → 검찰 **약식기소/불기소** | 경찰단계 |

**② 1~4호 지급 한도**

| 피해자 상해 등급 | 한도 |
|----------------|------|
| 사망 또는 1~3급 | 보험가입금액 한도 |
| 4급 이상 일반 부상 | MIN(보험가입금액, 3,000만원) |
| 다수 피해자 | 높은 한도 기준 1회 지급 |

**③ 5~6호 지급 한도 (경찰조사 단계)**

| 피해자 상해 등급 | 한도 |
|----------------|------|
| 사망 또는 1~3급 | MAX(보험가입금액, 5,000만원) |
| 중대법규위반으로 4~7급 | 3,000만원 |
| 중대법규위반으로 8~14급 | 가입금액 ≤5천만원 → 500만원 / 가입금액 >5천만원 → 1,000만원 |
| 다수 피해자 | 높은 한도 기준 1회 지급 |

---

### 제2조 (자동차 운전중 등의 정의)

**핵심 정의:**

| 용어 | 정의 |
|------|------|
| 자동차 운전중 | 도로여부·주정차여부·시동여부 불문, 운전석에 탑승하여 핸들 조작 가능한 상태 |
| 자동차 범위 | 승용·승합·화물·특수·이륜자동차 + 특정 건설기계 9종 |
| 구속 | 형사소송법상 피고인/피의자 신체 자유 제한 대인적 강제처분 |
| 기소 | 검사가 법원에 재판 청구하는 소송행위 |
| 약식기소 | 공판절차 없이 벌금·과료·몰수 형 청구 |
| 1사고 | 하나의 자동차 운전 중 교통사고 (경찰~대법원 전체 비용 합산하여 한도 적용) |

**중대법규위반 교통사고 정의:**
- 교통사고처리특례법 제3조 제2항 단서 해당 사고의 직접 결과로 타인이 부상등급을 받은 경우
- 불송치 결정서, 약식명령 공소장, 불기소이유고지서 등에 죄목/사고내용이 기재된 경우
- **단, 단서 7호·8호는 제외**

---

### 제4조 (보험금을 지급하지 않는 사유)

**절대적 면책:**
1. 피보험자 고의
2. 보험수익자 고의 (단, 일부 수익자는 나머지에 지급)
3. 계약자 고의
4. 전쟁·외란·혁명·내란·사변·폭동

**상대적 면책 (사고 원인이 아래인 경우):**
1. 사고 후 도주 (뺑소니)
2. 경기용·연습용·시험용 운전 중 사고
3. 건설기계가 **작업기계**로 사용 중 사고
4. **음주·무면허·약물** 상태 운전 중 사고 (도교법 43, 44, 45조 위반)

---

### 제7조 (보험금 청구 서류)

| 지급사유 | 필수 서류 |
|----------|-----------|
| 구속(1호) | 청구서 + 구속영장 사본 + 변호사 세금계산서 |
| 기소(2호) | 청구서 + 소장(공소장) + 변호사 세금계산서 |
| 약식→공판(3호) | 청구서 + 약식명령 공소장 + 변호사 세금계산서 |
| 정식재판(4호) | 청구서 + 정식재판 청구서 + 변호사 세금계산서 |
| 불송치(5호) | 청구서 + **불송치 결정서** + 변호사 세금계산서 |
| 불기소(6호) | 청구서 + **불기소이유고지서** 또는 **피의자처분결과 통지서** + 변호사 세금계산서 |

## 2. Agent 구현을 위한 활용 방안

### 2-1. 이 특약에서 Agent가 해야 할 일

```
[STEP 1] 청구 서류 분류
  제출 파일 중 아래 서류 존재 여부 확인:
    □ 보험금청구서
    □ 사고증명서 (불송치결정서 / 불기소이유고지서 / 공소장 등)
    □ 변호사 발행 세금계산서 (핵심: 실제 부담 금액 증빙)
    □ 교통사고 사실 확인원 또는 사고 경위서
    □ 피해자 진단서 (상해등급 판정을 위해)

[STEP 2] 지급 사유 판단
  사고증명서 유형 분류:
    "불송치 결정서"     → 5호 사유 해당 여부 검토
    "불기소이유고지서"  → 6호 사유 해당 여부 검토
    "공소장"            → 2호 또는 3호 사유 검토
    "구속영장"          → 1호 사유 확정
    "정식재판 청구서"   → 4호 사유 확정

[STEP 3] 피해자 상해등급 확인
  피해자 진단서에서 부상등급 추출:
    "전치 N주" → 등급 변환 테이블 적용
    "상해등급 N급" → 직접 적용
  → RULE-AUTO-DIN-001 부상등급표 참조

[STEP 4] 면책 사유 체크
  사고 경위에서 아래 키워드 탐지:
    "음주", "혈중알코올농도", "무면허", "도주"
  → 해당 시 즉시 부지급 (RULE-LAW-004)

[STEP 5] 보상 한도 계산
  5~6호 해당 + 피해자 1~3급 → MAX(가입금액, 5,000만원)
  청구된 변호사 비용 = 세금계산서 금액
  지급액 = MIN(청구금액, 적용한도)

[STEP 6] 선지급 여부 확인
  피보험자가 선지급 요청 시 → 가입금액의 70% 선지급
  잔여 30%는 사건 종결 후 지급
```

### 2-2. 이 특약의 복잡한 판단 포인트

```
판단 포인트 1: "중대법규위반 교통사고" 여부
  → 불송치결정서 또는 불기소이유고지서의 "죄목"에서
    교통사고처리특례법 제3조 제2항 단서 해당 조항 기재 확인
  → LLM이 자연어 문서에서 이를 추출해야 함
  → 예: "신호위반", "중앙선침범", "과속(제한속도 20km/h 초과)" → 해당
  → 예: "단순 접촉사고" → 미해당

판단 포인트 2: "불송치 vs 불기소" 구분
  불송치: 경찰이 검찰에 송치하지 않기로 결정 → 5호 사유
  불기소: 검찰이 기소하지 않기로 결정 → 6호 사유
  → 서류 제목으로 자동 분류 가능

판단 포인트 3: 1사고 1한도 적용
  동일 사고로 경찰 단계 + 이후 기소까지 진행 시
  → 전체 비용 합산하여 한도 1회 적용
  → 단계별로 중복 지급 불가
```

### 2-3. 현재 폴더 내 파일 활용 계획

```

     파일 시스템 접근 제한으로 직접 확인하지 못했습니다.

파일 확인 후 해야 할 것:
  1. 약관 버전 확인 (2023년 Hi2304 기준인지, 더 최신인지)

파일을 RAG에 넣을 때:
  → 청크 단위: 조항별 분리 (제1조, 제2조 등)
  → 메타데이터: {"문서_유형": "특약", "특약명": "변호사선임비용", "버전": "Hi2304"}
  → 검색 쿼리 예: "변호사 선임 비용 지급 조건" → 제1조 조항 히트 예상
```

## 3. 2025년 10월 개정 내용 (최신 변경사항)

```
변경 전 (Hi2304 기준):
  - 1사고 통합 한도 (심급 구분 없음)
  - 1~3급: 가입금액 한도

변경 후 (2025.10~ 예정):
  - 심급별 차등 지급:
    1심: 최대 2,000만원
    2심: 최대 2,000만원
    3심: 최대 1,000만원
    합산 총 5,000만원 한도

  - 선지급 비율: 70% 유지

영향:
  → 계약 체결일 기준으로 적용 약관 버전 결정
  → 2025.10 이전 계약 → 구약관 (Hi2304) 적용
  → 2025.10 이후 신규 계약 → 신약관 적용
  → Agent에서 계약일 기준 분기 처리 필요
```
