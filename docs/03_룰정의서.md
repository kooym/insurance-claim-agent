# 보험금 산정 룰 정의서 (PoC 범위)

> PoC 대상 담보: **실손의료비** / **입원일당** / **수술비**
> 기준 약관: 표준형 종합보험 (목업용, 실제 약관 구조 기반)

---

## 1. 공통 선행 점검 룰 (모든 담보 공통)

### RULE-COM-001: 계약 유효성 확인
```
조건: 사고일 기준 보험계약 상태 = "유효"
      보험료 납입 상태 = "정상" 또는 "납입유예기간 이내"
결과: 조건 미충족 시 → 부지급 (사유: 계약 실효)
```

### RULE-COM-002: 면책 기간 확인
```
조건: 사고일 >= 계약일 + 면책기간
      - 일반 질병: 계약일로부터 90일 이후
      - 암: 계약일로부터 90일 이후
      - 재해: 면책기간 없음
결과: 면책 기간 이내 → 부지급 (사유: 면책기간 해당)
```

### RULE-COM-003: 면책 사유 확인 (약관 제4조)
```
면책 사유 목록 (KCD 코드 기준):
  - 고의적 자해 / 자살 시도
  - 전쟁, 외란, 천재지변
  - 알코올 의존증 (F10), 알코올성 간질환 (K70.x)
  - 마약 관련 질환
  - 직업운동(프로선수) 중 사고
결과: 해당 시 → 부지급 (사유: 면책사유 해당, 조항 명시)
```

### RULE-COM-004: 중복 청구 확인
```
조건: 동일 입원건, 동일 수술건에 대해 이미 지급 이력 없음
결과: 중복 청구 확인 시 → 보류 (사유: 중복 청구 의심, 담당자 확인 필요)
```

## 2. 실손의료비 산정 룰

### RULE-SIL-001: 지급 대상 비용 산정
```
입력:
  - 진료비 영수증 항목별 금액
  - 국민건강보험 급여/비급여 구분

계산식:
  급여 본인부담금 지급액 = 급여 본인부담금 × 80%
  비급여 지급액 = 비급여 × 80%  (단, 도수치료/비급여주사/비급여MRI 등 특약 가입 여부 별도 확인)
  
  총 지급액 = (급여 본인부담금 + 비급여) × 80%
             단, 자기부담금 = MAX(총 의료비 × 20%, 최소 자기부담금 10,000원)
```

### RULE-SIL-002: 연간 한도 적용
```
조건: 해당 보험연도 내 누적 지급액 확인
한도: 입원 5,000만원 / 통원 30만원(회당), 180회/년
결과: 잔여 한도 내에서 지급
```

### RULE-SIL-003: 비급여 항목 특별 처리
```
도수치료: 특약 가입 시만 지급, 연 50회 한도
비급여 주사: 특약 가입 시만 지급, 연 50회 한도  
비급여 MRI/MRA: 특약 가입 시만 지급
```

## 3. 입원일당 산정 룰

### RULE-IND-001: 입원 인정 기준
```
조건:
  - 6시간 이상 입원 (당일 퇴원 포함 가능 여부는 특약 확인)
  - 의사 지시에 의한 입원
  - 입원확인서 + 진단서 모두 제출

인정 일수:
  - 입원일 + 퇴원일 = 2일 (입원당일, 퇴원당일 각 1일)
  - 예: 1/1 입원 ~ 1/5 퇴원 → 5일
```

### RULE-IND-002: 입원일당 지급액 계산
```
지급액 = 1일 지급금액 × 인정 입원일수

1일 지급금액: 계약 시 설정 (목업 기준: 30,000원/일)

연간 한도: 365일 / 1회 입원 최대 180일
```

### RULE-IND-003: 정신과/한방 입원 처리
```
정신과 입원 (F 코드): 특약 가입 여부 확인 후 지급
한방 입원: 원칙적 미지급 (단, 한방 특약 가입 시 지급)
```

## 4. 수술비 산정 룰

### RULE-SUR-001: 수술 해당 여부 판단
```
입력: 수술명 (한국 표준 수술 분류 코드 기준)
판단:
  수술 분류표 1종 ~ 5종 구분
  - 1종: 간단한 수술 (절개, 봉합 등)
  - 3종: 중등도 수술
  - 5종: 고난도 수술 (심장, 뇌 등)

지급액:
  수술비 = 수술 종류별 정액 지급
  - 1종: 100,000원
  - 2종: 300,000원
  - 3종: 500,000원
  - 4종: 1,000,000원
  - 5종: 2,000,000원
  (목업 기준 금액 — 실제 계약 금액으로 대체)
```

### RULE-SUR-002: 동일 수술 중복 지급 방지
```
조건: 동일 수술을 복수 담보로 청구 시
      → 중복 지급 불가, 최고액 1건만 지급
```

### RULE-SUR-003: 비수술적 시술 제외
```
제외 항목:
  - 내시경 단순 검사 (치료 없음)
  - 주사 투여
  - 물리치료
  → 이런 항목은 수술비 미해당, 실손으로 전환 처리
```

## 5. 최종 지급 결정 룰

### RULE-FIN-001: 지급/부지급/보류 판정 기준
```
지급:    모든 공통 룰 통과 + 담보 룰 산정 완료
부지급:  공통 룰 미통과 (면책, 유효성 등)
보류:    서류 미비 / 추가 확인 필요 / 이상 징후
일부지급: 한도 초과 / 비례 보상 적용
```

### RULE-FIN-002: 결과 출력 형식
```
{
  "claim_id": "청구번호",
  "decision": "지급|부지급|보류|일부지급",
  "payment_amount": 숫자,
  "applied_rules": ["RULE-COM-001", "RULE-IND-002", ...],
  "policy_clauses": ["제3조 제1항", "별표2", ...],
  "reason": "지급 사유 또는 부지급 사유 텍스트",
  "required_docs": ["추가 필요 서류 목록 (보류 시)"],
  "reviewer_flag": true/false  # 담당자 검토 필요 여부
}
```

## 6. 서류 및 신뢰도 검증 룰

### RULE-DOC-CHECK: 서류 완비 확인
```
조건: 청구 담보에 필요한 필수 서류가 모두 제출되었는지 확인
  - 입원일당: 청구서 + 진단서 + 입원확인서
  - 실손의료비: 청구서 + 진단서 + 진료비영수증 (+ 세부내역서 권장)
  - 수술비: 청구서 + 진단서 + 수술확인서
결과: 필수 서류 미제출 시 → 보류 (서류보완요청)
```

### RULE-CONF-001: 파싱 신뢰도 검증
```
조건: 서류별 파싱 confidence 최솟값 < 0.6 (임계값)
결과: FLAGGED — 담당자 검토 필요 (파싱 신뢰도 낮음)
  → 자동 판정 계속하되 reviewer_flag 부여
```

### RULE-CHRONIC-ONSET: 만성질환 발병일 처리
```
조건: 발병일이 "수년 전부터 만성 경과", "불명", "미상" 등 비정형 표현
처리: 
  1. 입원일(최초 치료일)을 면책기간 판정 기준일로 사용 (보험업계 표준)
  2. FLAGGED — 기왕증 여부 및 고지의무 확인 필요
  3. 원본 발병일 표현을 evidence에 보존
근거: 상법 제651조 (고지의무), 금감원 분쟁조정 판례
```

## 7. 룰 우선순위

```
1순위: RULE-COM (공통 룰) — 하나라도 실패 시 즉시 부지급
2순위: RULE-FIN-001 (판정)
3순위: 담보별 룰 (SIL / IND / SUR)
4순위: 한도 및 중복 확인
```
