# 입원일당 지급기준 (Agent 구현용)

> **출처 및 법적 근거**
> - 현대해상 영업용자동차보험 약관 제4조 (입원의 정의)
> - 실손의료보험 표준약관 입원일당 담보 조항
> - 금융감독원 실손의료보험 보험금 지급사유 조사 기준 (2024.05)
>
> **✅ Phase 2 반영**: `rule_engine.py::rule_ind()` 에서 **입원 면책일수**(질병 4일/재해 1일) 차감 적용됨.
> 가입일 기준 면책기간은 COM-002에서 별도 처리.

---

## 1. 입원 인정 기준

### 1.1 입원의 정의 (약관 제4조)

| 조건 | 내용 |
|------|------|
| 의사 지시 | 의사의 입원 지시에 의한 입원 |
| 체류 시간 | 6시간 이상 의료기관 체류 (당일 퇴원 포함) |
| 기록 | 진료기록부 상 입원 기록 존재 |
| 필요성 | 자택 등에서 치료 불가한 사유 |

### 1.2 입원 일수 계산

```
입원 일수 = (퇴원일 - 입원일 + 1)
```

- 입원 당일 1일, 퇴원 당일 1일 각각 산정
- 예: 11/10 입원 ~ 11/14 퇴원 → **5일**

## 2. 면책일수 (입원 발생 후 차감)

> **코드 반영** — `rule_ind()` 에서 `payable_days = max(0, hospital_days - 4 또는 -1)` 적용

### 2.1 질병 입원

| 항목 | 값 | 비고 |
|------|-----|------|
| 면책일수 | **4일** | 입원 발생 후 처음 4일은 지급 제외 |
| 지급 대상 일수 | `MAX(0, 입원일수 - 4)` | 5일 이상 입원 시 1일부터 지급 |

**예시**
- 3일 입원 → 지급 0일 (3 - 4 = -1 → 0)
- 5일 입원 → 지급 1일 (5 - 4 = 1)
- 10일 입원 → 지급 6일 (10 - 4 = 6)

### 2.2 재해 입원

| 항목 | 값 | 비고 |
|------|-----|------|
| 면책일수 | **1일** | 입원 발생 후 처음 1일은 지급 제외 |
| 지급 대상 일수 | `MAX(0, 입원일수 - 1)` | 2일 이상 입원 시 1일부터 지급 |

**예시**
- 1일 입원 → 지급 0일
- 3일 입원 → 지급 2일 (3 - 1 = 2)
- 10일 입원 → 지급 9일 (10 - 1 = 9)

### 2.3 질병 vs 재해 구분

| 구분 | KCD 코드 | 판별 기준 |
|------|----------|----------|
| 재해 | V, W, X, Y 계열 | 외인 손상 및 중독 (교통사고, 낙상 등) |
| 질병 | 그 외 | 내인성 질환 (감염, 암, 수술 등) |

## 3. 연간 한도 및 최대 지급일수

### 3.1 1회 입원 최대 지급일수

| 한도 | 값 | 비고 |
|------|-----|------|
| 1회 입원 | **180일** | 동일 입원 건당 상한 |
| 연간 누적 | **365일** | 보험연도 기준 |

### 3.2 연간 한도 체크 로직

```
잔여일수 = 365 - 해당_보험연도_누적입원일수
인정_입원일수 = MIN(계산된_지급대상일수, 잔여일수, 180)
```

## 4. 재입원 사이클 기준

### 4.1 동일 입원 vs 별도 입원

| 조건 | 판정 |
|------|------|
| 퇴원 후 **14일 이내** 재입원 | **동일 입원**으로 간주 (연속 입원) |
| 퇴원 후 **15일 이상** 경과 후 재입원 | **별도 입원** (새 면책일수 적용) |

### 4.2 동일 입원 시 처리

- 입원일수 합산 (이전 퇴원일 ~ 이번 입원일 연속)
- 면책일수는 **최초 입원일** 기준 1회만 차감
- 1회 입원 180일 한도는 합산 일수에 적용

### 4.3 별도 입원 시 처리

- 면책일수 **재적용** (질병 4일 / 재해 1일)
- 1회 입원 180일 한도 **별도** 적용
- 연간 365일 한도는 누적 적용

## 6. 특수 입원 처리

### 6.1 정신과 입원 (KCD F코드)

- **정신질환입원특약** 가입 여부 확인
- 미가입 시 → 부지급 (COM-003 조건부면책 또는 IND-003)

### 6.2 한방 입원

- **한방입원특약** 가입 여부 확인
- 미가입 시 → 부지급

### 6.3 요양병원 입원

- 기본 입원일당 적용 (요양병원 특약 별도 확인)
- FRD-013 의학적 필요성 검토 플래그

## 7. Agent 구현 시 참조

### rule_engine.py::rule_ind() 연결

| Phase | 구현 내용 |
|-------|----------|
| **현재** | hospital_days 그대로 사용, 면책일수 미차감 |
| **Phase 2** | `claim_nature`(질병/재해)에 따라 4일/1일 차감 후 지급일수 산정 |
| **Phase 2** | 재입원 14일 사이클 로직 추가 (claims_history 연계) |

### 계산식 (Phase 2 목표)

```
# 질병
payable_days = MAX(0, hospital_days - 4)

# 재해
payable_days = MAX(0, hospital_days - 1)

# 최종
payable_days = MIN(payable_days, 180, 365 - annual_used_days)
benefit_amount = payable_days × daily_benefit
```
