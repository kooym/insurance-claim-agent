## 2. 1~5종 분류 정의

### 2.1 1종 — 단순·경미 수술

| 정의 | 내용 |
|------|------|
| 범위 | 단순 절개, 봉합, 발치, 경미한 조직 절제 |
| 특징 | 국소마취 또는 단순 시술로 완결 |
| 예시 | 백내장 인공수정체삽입, 치핵, 포경, 발톱 발치, 피부 낭종, 지방종 단순 절제 |

### 2.2 2종 — 중간 수술 (복강경·경피적)

| 정의 | 내용 |
|------|------|
| 범위 | 복강경·내시경·경피적 시술 |
| 특징 | 개복 없이 최소 침습 |
| 예시 | 담낭 복강경, 난소 낭종 절제, 갑상선 RFA, 자궁 HIFU, 편도·아데노이드 절제 |

### 2.3 3종 — 표준 개복·복강경 수술

| 정의 | 내용 |
|------|------|
| 범위 | 일반적인 개복 또는 복강경 수술 |
| 특징 | 장기·조직 절제 또는 재건 |
| 예시 | 충수 복강경/개복, 유방 종양 절제, 갑상선 부분절제, 자궁 근종 절제, 추간판 절제 |

### 2.4 4종 — 대형 수술

| 정의 | 내용 |
|------|------|
| 범위 | 장기 전절제, 광범위 절제 |
| 특징 | 암·심각한 질환에 대한 근치적 수술 |
| 예시 | 유방 전절제, 자궁 전절제, 갑상선 전절제(악성), 대장 절제, 위 부분절제, 심장 박동기 삽입 |

### 2.5 5종 — 고위험·중대 수술

| 정의 | 내용 |
|------|------|
| 범위 | 뇌·심장·폐·신장 등 주요 장기 수술 |
| 특징 | 고위험·장시간 수술 |
| 예시 | 두개골 절개 뇌수술, 관상동맥 우회술, 심장판막 치환술, 폐 절제, 위 전절제, 신장 절제 |

## 7. Agent 구현 시 참조

### rule_engine.py::rule_sur() 연결

- `get_surgery_class(surgery_code, surgery_name, kcd_code)` → `surgery_classification.json` 조회
- `class` → 1~5종, `_benefit` 테이블로 지급액 산정
- KCD 기반: `kcd_to_surgery_codes` 역인덱스로 후보 추론 (복수 후보 시 담당자 확인)

### 분류표 확장 시

- `surgery_classification.json`의 `surgery_map`에 항목 추가
- `name_to_code_index`, `kcd_to_surgery_codes` 동기화 필요
