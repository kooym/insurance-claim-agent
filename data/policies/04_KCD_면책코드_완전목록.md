## 2. 조건부면책 (Conditional Exclusions)

**특약 가입 여부** 또는 **담당자 확인** 후 판단.

### 2.1 정신질환 (F계열)

| KCD 범위 | 진단명 | 특약 |
|----------|--------|------|
| F00 ~ F09 | 치매·기질성 정신장애 | 정신질환입원특약 |
| F20 ~ F29 | 조현병·망상장애 | 정신질환입원특약 |
| F30 ~ F39 | 기분장애 (우울·조증) | 정신질환입원특약 |
| F40 ~ F48 | 신경증·스트레스 관련 | 정신질환입원특약 |
| F50 ~ F59 | 행동장애 (섭식장애 등) | 정신질환입원특약 |
| F60 ~ F69 | 성격장애 | 정신질환입원특약 |
| F70 ~ F79 | 지적장애 | 정신질환입원특약 |
| F80 ~ F89 | 발달장애 | 정신질환입원특약 |
| F90 ~ F99 | 아동기 행동장애 등 | 정신질환입원특약 |

**처리 로직**
```
IF KCD ∈ F계열 (F10, F11~F19 제외)
  → CHECK_COVERAGE("정신질환입원특약")
  → 미가입 시 FAIL (조건부면책)
  → 가입 시 PASS (단, 입원일당 IND-003 별도 확인)
```

### 2.2 한방 치료

| 조건 | 처리 |
|------|------|
| 한방 입원 | 한방입원특약 가입 여부 확인 |
| 한방 통원 | 한방통원특약 가입 여부 확인 |

**확인 방법**: 원무과 서류에 'KM', '한의원', '한방' 기재 여부

### 2.3 선천기형 (Q계열)

| KCD 범위 | 진단명 | 비고 |
|----------|--------|------|
| Q00 ~ Q99 | 선천기형·선천이상 | 계약 전 존재 여부 확인 |

**처리**: 기왕증(선천성) 여부 → 청약서 고지의무 연계, MANUAL_REVIEW

### 2.4 미용 (Z41.1) — 치료 혼재 시

| 상황 | 처리 |
|------|------|
| 순수 미용 | 부지급 |
| 치료+미용 동시 | FRD-005 담당자 검토, 치료 부분만 분리 산정 |

## 5. Agent 구현 시 참조

### rule_engine.py::rule_com_003() 연결

- `check_kcd_exclusion(kcd_code)` → `kcd_exclusion_map.json` 조회
- **absolute_exclusions**: prefix 매칭 → 즉시 FAIL
- **conditional_exclusions**: `action: CHECK_COVERAGE` → 특약 확인 후 판단

### kcd_exclusion_map.json 보강 (T16)

- F계열 전체 코드 명시 (현재 `conditional_exclusions.mental_disorder.codes`에 포함)
- Z41.1 `cosmetic` 항목 (현재 `absolute_exclusions`에 포함)
- Q계열 `congenital` 조건부면책 추가
- 특약연계 처리 로직 주석 보강
